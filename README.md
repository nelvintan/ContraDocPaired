# About
- This repository contains the ContraDocPaired dataset.
- Positive data (where an inconsistency exists) are in the file `positive_data.xlsx` and negative data (where no inconsistency exists) are in the file `negative_data.xlsx`.
- There are exactly 2 inconsistencies in each positive datapoint.
- This dataset was constructed from ContraDocPaired (https://arxiv.org/abs/2311.09182).
- Code to construct this dataset is in `TOFILL`.
